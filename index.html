「Webコンテンツ作成」の成果

本テーマの学習目的

  • HTMLによってWebページの構造を記述する基本的な方法がわかる
  • CSSによってWebページの見た目を設定する基本的な方法がわかる
  • JavaScriptによってWebページのインタラクティブな処理を記述する基本的な方法がわかる
  • HTML,CSS,JavaScript•HTML,CSS,JavaScript の構造リファレンスから,構造の書式や使用方法の情報を入手できる
  • HTML,CSS,JavaScrupt,画像・音声・動画ファイルを組み合わせて簡単なWebページが作成できる

以上のこれらを学習目標として取り組み、力を身につけることを目的とする。

プロジェクト課題について

プロジェクト課題はとても複雑で難しかったです。一つ一つを見れば授業で習った内容ですが組み合わせて作るのが難しいです。複雑に組み合わせたり、出力したりするのが大変で全然上手くいきませんでした。資料を何度も読み直したり他のサイトも探したりしました。また、検索する際はリストからお店の種類を選んで見られるようにしました。ページはもっと装飾を綺麗にできたのではないかと反省しています。時間も計画を立ててやればしっかり確保できたと思うので計画性を持って行動することも考えなければいけないと思いました。

数当てゲーム

数当てゲームは1~10の中からランダムの数値を答えにしてそれを当てるというものです。3回まで回答して4回目以降は正解の発表とこれ以上は答えても意味ないことを教えます。また、ヒントとして予想が答えより大きいか小さいか教えてくれます。シンプルな内容ですがプログラミングをしてみるとなかなか苦戦しました。テキストボックスを作る、予想回数に応じた動きの変更などやることが多かったです。また改良案としては答えが10で1回目の予想が5、2回目の予想が7の時1回目のヒントと違い「もっと大きい数です。」と言ったり、1回目の予想が5、2回目の予想が3の時「大きい数とヒントを出しましたよ？」と言った感じでバリエーションを増やしたらさらに面白いかなと思いました。

毎回の実験で作成したページ

第1回 WWW と HTML

第1回はwwwとHTMLについて学びました。wwwはWebページを提供する仕組みのことです。HTMLはWebページの構造を記述するための書式です。HTMLによってWebページに掲載するコンテンツと,ページの基本的なレイアウトを設定できます。

演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

第2回はCSSについて学びました。CSSはページを装飾する記述です。CSSを使えば色変えたり、大きさを変えたり、フォントを変えたり、線を引いたりと指定した文字や箇所の見た目を自由に変えることが可能になります。

練習問題2-3:ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

第3回はJavaScriptについて学びました。JavaScriptはWebページ上で複雑な機能を実現するプログラムです。Webページを開いた直後や、ボタンを押した時にプログラムが実行されます。これによりページの個性がより大きく出るようになります。

練習3-6:オブジェクトの参照

第4回 DOM と JavaScript

第4回はDOM(Document Object Model)について学びました。DOMはWebコンテンツを表現するデータ構造です。DOMを使用することにより、JavaScriptなどのプログラムによって、Webコンテンツの内容を読み取ったり、コンテンツに変更を加えることができます。

練習4-2から練習4-5

第5回 イベント処理

第5回はイベント処理について学びました。イベントとは簡単に言えば動きの動きをスイッチとしてプログラムを動作させるのがイベント処理です。例を挙げるとマウスでクリックしたり、キーボードのキーを押したりすると動作します。

情報メディア実験「Web コンテンツ作成」の感想

この授業のWebコンテンツ作成ではさまざまなことを学びました。まず初めに苦労したところはページのレイアウトを考えることです。色を変えたりページの内容のまとめ方などを考えました。他の人と似たページにならないようにと思いあまり使われないような色の組み合わせにしてみました。ページのレイアウトはシンプルな縦並びにしました。理由は自分が一番見やすい形だと思ったからです。

回ごとの感想

第1回はHTMLの基本的な部分をやりましたが気をつけたのはどこからどこまでが囲われているかです。二重になったりする部分がどうなっているか気をつけなければ思ったとこに文字がなかったり大きさや太字が違ったりしたのが大変でした。

第2回はCSSを使ってページに装飾をしました。グラデーションにしたり文字の背景をつけたりしました。自由に色をつけたりするより見本と全く同じものを作る方が難しいと感じました。

第3回はコンソールに指定のものを出力しました。また、JavaScriptのfor-ofの文章について学んだ。どこにその情報が入っているのかに気をつけて取り出しました。一気に答えに辿り着こうとするのではなくしまってある場所や取り出し方を一つずつ確認しながら進んでいくことを意識しました。

第4回はDOMについて学びました。DOMはHTMLに変更を変えることができます。DOMはJavaScriptで書くのでHTMLを直接変えるのと難易度が違い難しかったです。思ったよりも想像した通りの動きをしてくれませんでした。回ごとの内容では一番難しかったと思います。

第5回はイベント処理を学びました。クリックしたりカーソルを動かすことで色が変わるなどの動作が見られますが今までの回で一番楽しく、内容もわかりやすかったです。イベントの種類が多く全て使いこなそうとするのは難しいと思いました。

まとめ

Webコンテンツは自分で作ってみるとかなり大変でした。インターネットでいろんなサイトを見ますがそれらを作ろうと考えるだけで恐ろしいです。さらに定期的に更新したりしているサイトはなおのことすごいと感じました。ですが授業で習った以上そのうち自分で何か新しくサイトを作ってみるのも面白いと思います。この授業で習っていないこともあるのでそれらを自分で学んでやってみたいです。また、授業で触れたものを全て使いこなせたわけではないので今後またサイトを作るときは使いきれなかったことも使っていきたいです。今回作ったサイトももっと色が変わる仕組みを作る、レイアウトを工夫できたのではないかなどの反省点もあります。次に作るときは今回のものより凝ったものを作りたいです。
