「Webコンテンツ作成」の成果

本テーマの学習目的

  • HTMLによってWebページの構造を記述する基本的な方法がわかる
  • CSSによってWebページの見た目を設定する基本的な方法がわかる
  • JavaScriptによってWebページのインタラクティブな処理を記述する基本的な方法がわかる
  • HTML,CSS,JavaScript•HTML,CSS,JavaScript の構造リファレンスから,構造の書式や使用方法の情報を入手できる

    • HTML,CSS,JavaScrupt,画像・音声・動画ファイルを組み合わせて簡単なWebページが作成できる

以上のこれらを学習目標として取り組み、力を身につけることを目的とする。

毎回の実験で作成したページ

第1回 WWW と HTML

第1回はwwwとHTMLについて学びました。wwwはWebページを提供する仕組みのことです。HTMLはWebページの構造を記述するための書式です。HTMLによってWebページに掲載するコンテンツと,ページの基本的なレイアウトを設定できます。

演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

第2回はCSSについて学びました。CSSはページを装飾する記述です。CSSを使えば色変えたり、大きさを変えたり、フォントを変えたり、線を引いたりと指定した文字や箇所の見た目を自由に変えることが可能になります。

練習問題2-3:ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

第3回はJavaScriptについて学びました。JavaScriptはWebページ上で複雑な機能を実現するプログラムです。Webページを開いた直後や、ボタンを押した時にプログラムが実行されます。これによりページの個性がより大きく出るようになります。

練習3-6:オブジェクトの参照

第4回 DOM と JavaScript

第4回はDOM(Document Object Model)について学びました。DOMはWebコンテンツを表現するデータ構造です。DOMを使用することにより、JavaScriptなどのプログラムによって、Webコンテンツの内容を読み取ったり、コンテンツに変更を加えることができます。

練習4-2から練習4-5

第5回 イベント処理

第5回はイベント処理について学びました。イベントとは簡単に言えば動きの動きをスイッチとしてプログラムを動作させるのがイベント処理です。例を挙げるとマウスでクリックしたり、キーボードのキーを押したりすると動作します。
